# Well-being & Confidence Images

This directory contains images for the "Well-being & Confidence" section of the Menstrual Health module.

## Required Images

Place the following images in this directory:

- `mood_changes.png` - Image for understanding mood changes
- `managing_cramps.png` - Image for managing cramps and discomfort
- `self_confidence.png` - Image for building self-confidence
- `staying_active.png` - Image for staying active and energized
- `nutrition.png` - Image for nutrition and well-being
- `stress_management.png` - Image for stress management and relaxation
- `sleep_rest.png` - Image for sleep and rest

## Image Specifications

- Format: PNG (recommended) or JPG
- Recommended size: 800x600px or similar aspect ratio
- The images will be displayed at 200px height with contain resize mode

## Notes

If images are not found, the app will display a placeholder with instructions to add the image.
