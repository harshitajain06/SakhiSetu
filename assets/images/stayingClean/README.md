# Staying Clean Images

This directory contains images for the "Staying Clean" section of the Menstrual Health module.

## Required Images

Place the following images in this directory:

- `daily_hygiene.png` - Image for daily hygiene practices
- `sanitary_products.png` - Image for choosing sanitary products
- `disposal.png` - Image for proper disposal of sanitary products
- `prevent_infections.png` - Image for preventing infections
- `bathing.png` - Image for bathing and showering
- `managing_odor.png` - Image for managing odor

## Image Specifications

- Format: PNG (recommended) or JPG
- Recommended size: 800x600px or similar aspect ratio
- The images will be displayed at 200px height with contain resize mode

## Notes

If images are not found, the app will display a placeholder with instructions to add the image.
